# Security Policy

## Reporting a vulnerability

If you believe you have found a security issue in `ogoron-actions`, do not open a
public GitHub issue.

Report it privately to:

- `info@ogoron.com`

Please include:

- the affected action and ref or version
- a minimal workflow snippet or reproduction setup
- the expected impact
- clear reproduction steps
- any relevant sanitized logs or artifacts

## Disclosure expectations

Please give Ogoron a reasonable opportunity to investigate and remediate the
issue before public disclosure.

## Scope

This policy covers:

- the actions in this repository
- the repository examples and workflow glue maintained here

This policy does not directly cover:

- vulnerabilities in GitHub Actions runners or GitHub itself
- vulnerabilities in third-party actions used by customer workflows
- security issues inside customer repositories, infrastructure, or secrets handling
